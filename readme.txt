1. Filevalue.py is the code to assign meta data (external criticality, role access and number of access) to a file
2. fileMaker.py is to generate the files of size 2MBs.
3. nameNode.py build the Hadoop alike name node.
4. main.py is the driver program which runs the code.
5. To run the setup, first of all run fileValue.py and then run main.py
